## Repository Overview

This is a documentation repository containing study notes for the Snowflake SnowPro Core certification exam (COF-C02/COF-R02). The content is published to GitHub Pages at https://snowflake-study-group-atlanta.github.io/snowprocore/.

## Structure

The repository consists of Markdown files organized by exam domain:
- `snowflake_ai_data_cloud_features_and_architecture.md` - Domain 1 (24%)
- `account_access_and_security.md` - Domain 2 (18%)
- `performance_and_cost_optimization_concepts.md` - Domain 3 (16%)
- `data_loading_and_unloading.md` - Domain 4 (12%)
- `data_transformations.md` - Domain 5 (18%)
- `data_protection_and_data_sharing.md` - Domain 6 (12%)

## Content Conventions

- Each domain file follows the official exam guide structure with hierarchical headings
- Key concepts are highlighted using blockquote callouts (>) for quick review
- Callouts summarize important facts, comparisons, and exam-relevant distinctions
- Content focuses on conceptual understanding rather than step-by-step procedures

## No Build System

This is a pure Markdown documentation repository with no build tools, package managers, or test suites. GitHub Pages serves the Markdown files directly.
